# OBD2 Monitor на ESP32

## Описание
Проект для чтения и обработки данных OBD2 через CAN-интерфейс с использованием микроконтроллера ESP32.

## Функционал
- Чтение стандартных параметров (PID)
- Расчет расхода топлива
- Детектирование состояния двигателя
- Хранение данных поездки
- Вывод информации через UART

## Установка
1. Установите ESP-IDF v5.5
2. Клонируйте репозиторий
3. Соберите проект: `idf.py build`
4. Прошейте на устройство: `idf.py -p PORT flash monitor`

## Поддерживаемые PID
| PID | Описание |
|-----|----------|
| 0x0C | Обороты двигателя |
| 0x2F | Уровень топлива |
| 0x31 | Пробег |
