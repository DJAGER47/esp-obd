# OBD2 Monitor на ESP32

## Описание
Проект для чтения и обработки данных OBD2 через CAN-интерфейс с использованием микроконтроллера ESP32.

## Функционал
- Чтение стандартных параметров (PID)
- Расчет расхода топлива
- Детектирование состояния двигателя
- Хранение данных поездки
- Вывод информации через UART

## Установка
1. Установите ESP-IDF v5.5
2. Клонируйте репозиторий
3. Соберите проект: `idf.py build`
4. Прошейте на устройство: `idf.py -p PORT flash monitor`

## Отладка unit-тестов

Для отладки unit-тестов в среде VS Code:

1. Убедитесь, что установлены необходимые зависимости:
   - CMake 3.22 или выше
   - GCC/Clang с поддержкой C++17
   - GDB

2. Откройте проект в VS Code

3. Перейдите на вкладку "Run and Debug" (Ctrl+Shift+D)

4. Выберите нужную конфигурацию запуска:
   - "Debug Unity App" для запуска с обычной сборкой
   - "Debug Unity App (Debug Build)" для запуска с отладочной сборкой

5. Нажмите зеленую кнопку запуска или F5 для начала отладки

## Поддерживаемые PID
| PID | Описание |
|-----|----------|
| 0x0C | Обороты двигателя |
| 0x2F | Уровень топлива |
| 0x31 | Пробег |
